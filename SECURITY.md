# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Cove, please report it responsibly.

**Do not open a public issue.** Instead, email **micheletti.emanuele@hotmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Any relevant logs or screenshots

You should receive an acknowledgment within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

Security issues we care about include (but are not limited to):

- Credential leakage (passwords, tokens, keys)
- Unauthorized access to database connections
- Code injection via query input
- Keychain data exposure
